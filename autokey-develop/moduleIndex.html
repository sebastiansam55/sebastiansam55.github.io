scripting API Documentation

Module Index

  • scripting - This package contains the scripting API.
    • scripting.clipboard_gtk - GtkClipboard Functions
    • scripting.clipboard_qt - QtClipboard Functions
    • scripting.common - Contains common functions for both Qt and Gtk versions
    • scripting.dialog_gtk - Class for creating Gtk Window dialogs
    • scripting.dialog_qt - Class for creating Qt dialogs
    • scripting.engine - Engine backend for Autokey
    • scripting.highlevel - Highlevel scripting API, requires xautomation to be installed
    • scripting.keyboard - Keyboard Functions
    • scripting.mouse - Mouse functions, see also highlevel for mouse move and click functions using xte from xautomation
    • scripting.system - Provides access to run system commands through subprocess and basic file creation
    • scripting.window - Basic window management. Requies wmctrl to be installed.
